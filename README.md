# rock-paper-scissors-cpp
A simple Rock Paper Scissors game, while learning c++.

## DEVELOPMENT
* A C++ Compiler
* [Make](https://www.gnu.org/software/make/)

### COMPILE

To build:
`make`

To build and run executable:
`make run`
